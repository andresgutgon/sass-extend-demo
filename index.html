The problem:

In DEMO 1 I only want to change the color of .base-button under .blue-parent. But...

SCSS code: extending from .base-button


.base-button {
  background-color: #f0f0f0;
  border-radius: 20px;
  border: 1px solid #ccc;
  color: #333;
  padding: 10px;
  font-size: 13px;
  margin: 0 10px 0 0;
}

.base-button--in-red {
  @extend .base-button;
  background-color: #AD210E;
  color: white;
  border-color: #AD210E;
}

// I only want my base button in blue
.blue-parent {
  .base-button {
    color: white;
    background-color: blue;
  }
}

... blue style is applied to both buttons. This was unexpected to me

Generated CSS code in DEMO 1


// Here is the problem.
